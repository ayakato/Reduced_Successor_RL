## ------ About this file ------ ##
This file contains codes for running simulation and generating figures on our manuscript "Reduced Successor Representation Potentially Interferes with Cessation of Habitual Reward-Seeking".
The manuscript is the revised version of the previous one ("Difficulty in Cessation of Undesired Habits: Goal-Based Reduced Successor Representation and Reward Prediction Errors"), and is available on bioRxiv (URL on here).


## ------ How to generate data and figures ------ ##
First, please run the "**.py" files to generate csv files in corresponding empty file.
Then, running "Figure_Plot.R" in "Figure" directory will give you figures in corresponding empty file.
